# ClinicManager 快速上手指南

> 面向基层医生的诊所管理系统。5 分钟学会日常使用。

---

## 界面布局

```
┌──────────┬──────────────────────────────────────┐
│          │  诊所管理系统                   设置  │  ← 顶部栏
│  📋 接诊 ├──────────────────────────────────────┤
│  💊 药品 │                                      │
│  📖 方剂 │         (内容区域)                    │
│  ─────── │                                      │
│  ⚙ 系统 │                                      │
│          │                                      │
│  ← 侧边栏│                                      │
└──────────┴──────────────────────────────────────┘
```

侧边栏可收纳（点击侧边栏边缘的箭头），收起来后给内容区更多空间。

## 日常流程：接诊 → 开药 → 打印

### 第一步：新建接诊

1. 点击侧边栏 **📋 接诊**，进入接诊列表
2. 点击左下角 **新建接诊** 按钮
3. 填写：**患者姓名、性别、年龄、诊断**
   - 年龄可以手动填，也可以填出生日期自动计算

### 第二步：开处方

在接诊编辑页下方，有一个处方表格：

1. **选择处方类型**——点击"西药"或"中药"标签，决定当前处方类型
2. **添加药品**
   - 鼠标点击处方表格的**药品名称格**，或按 `Enter` 键
   - 弹出搜索框，输入药品名称关键词搜索
   - 选中后，规格、剂型、单价自动填入
3. **填写用量**
   - 用量：输入单次用量值（如 2）
   - 单位：选择单位（片、粒、mg、g 等）
   - 频次：选择每日几次（如 每日3次）
   - 疗程：输入天数（如 7天）
4. **分组**（可选）
   - 如果多条药品的用法频次相同，在**组号**列填上相同数字
   - 同组药品自动同步用法频次疗程（最后写入的覆盖前面）
   - 点击组号可按组排序

**全部使用键盘的操作流（推荐）：**

```
Enter  → 弹出药品搜索框（药品名列）
Tab    → 跳到下一列
Enter  → 弹出频次/疗程下拉菜单（用量/频次/疗程列）
```

全程不用碰鼠标。

### 第三步：保存与打印

1. 填写完成后，点击 **保存**
2. 点击 **打印预览** 查看 A5 处方效果
3. 确认无误后，点击 **打印** 或 **导出 PDF**

## 常用功能

### 查看患者历史

在接诊列表选中一条记录，右侧会显示患者信息卡片。点击卡片上的 **历史记录**，可查看该患者的所有历史就诊记录。

历史病历是只读的——内容不可修改，确保医疗数据真实可追溯。需要复用旧处方时，使用 **调用到处方** 功能。

### 使用方剂库（中药）

1. 切换到中药处方
2. 在处方表格上方点击 **方剂库**
3. 浏览或搜索方剂（支持按分类、按功效、按名称搜索）
4. 选中方剂，点击 **展开到处方**
5. 所有药品自动填入处方表格，只需调整用量即可

### 管理药品目录

在侧边栏点击 **💊 药品目录**：
- **新建药品**：填写名称、规格、剂型、单价等信息
- **导入药品**：支持 .xls / .xlsx / .csv 格式批量导入
- **编辑/删除**：选中药品后修改，删除为软删除（可恢复）

### 备份与恢复

在侧边栏点击 **⚙ 系统管理** → **数据管理**：
- **手动备份**：立即创建数据快照
- **恢复**：从备份列表中选择一个时间点恢复
- **导出数据**：CSV 格式导出接诊数据

系统每天启动时也会自动备份。

## 遇到问题

- **打印不对**：检查打印机纸张设置是否为 A5
- **药品搜索不到**：先去药品目录确认药品已添加
- **数据异常**：在系统管理中尝试恢复最近一次备份
